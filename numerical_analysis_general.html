📖 Course Overview

CHAPTER 1 Error Estimation

Numerical Analysis provides approximate (not exact) solutions to mathematical problems. Error estimation is the study of how far an approximate value deviates from the true value.

The fundamental relationship: Error = True Value − Approximate Value
  • Types of error: Absolute, Relative, Percentage
  • Sources: Inherent, Rounding, Truncation
  • Error propagation through functions
CHAPTER 2 Non-Linear Equations

Finding the roots of f(x) = 0 where f(x) contains exponential, logarithmic, trigonometric, or polynomial terms of degree > 2. These cannot be solved by simple algebra, requiring iterative numerical methods.

A root (zero) of f(x) = 0 is a value α such that f(α) = 0. Geometrically, it is where the curve y = f(x) crosses the x-axis.
  • 4 iterative methods: Bisection, False Position, Newton-Raphson, Secant
  • Root location using Intermediate Value Theorem
  • Convergence criteria and stopping conditions

📌 Key Definitions
Algebraic Equation

Contains only polynomial terms.
e.g. 3x³ − 2x² − x − 5 = 0

Transcendental Equation

Contains exp, log, or trig functions.
e.g. xe^x − 1 = 0  |  tan(x) = x

Direct Method

Gives exact value in finite steps. Only available for polynomials up to degree 4.

Iterative Method

Starts with approximation(s), generates sequence x₀, x₁, x₂,... converging to the root.

📏 Chapter 1: Basic Concepts in Error Estimation

🔍 Types of Errors
Inherent Error

Present in the problem before computation begins. Arises from approximate input data or limitations of mathematical modeling.

Rounding Error

Caused by rounding numbers during computation.
Example: 13.658 rounded → 13.66
Error = 13.66 − 13.658 = 0.002

Truncation Error

Caused by replacing an infinite process with a finite one.
Example: 13.658 truncated → 13.65
Error = 13.658 − 13.65 = 0.008

1 Absolute Error (Eₐ)
Formula
Eₐ = |X − X̃|

Variables:

  • X — True (exact) value
  • X̃ — Approximate value
  • | · | — Absolute value (result is always non-negative)
The absolute error measures the magnitude of the deviation between the true and approximate value, without regard to direction.
Worked Example 1

True value X = 3.14159, Approximate value X̃ = 3.14

  1. Apply formula: Eₐ = |X − X̃|
  2. Substitute: Eₐ = |3.14159 − 3.14|
  3. Evaluate: Eₐ = |0.00159|
  4. Result: Eₐ = 0.00159
Worked Example 2 — From Lecture Notes

Computed value: 2.571282, True value (∛17): 2.571332

  1. Eₐ = |2.571332 − 2.571282|
  2. Eₐ = 0.000050
  3. Result: Eₐ = 0.00005 → Correct to 4 decimal places ✓
2 Relative Error (Eᵣ)
Formula
Eᵣ = Eₐ / |X| = |X − X̃| / |X|

Purpose: Expresses the error as a fraction of the true value. Relative error is independent of the units of measurement (noted in your lecture notes).

Worked Example 1

True = 3.14159, Approx = 3.14, Eₐ = 0.00159

  1. Eᵣ = Eₐ / |X| = 0.00159 / 3.14159
  2. Result: Eᵣ ≈ 0.000506
Worked Example 2 — Error Propagation (From Lecture Notes)

Given: u = 4x²y³/z⁴, errors in x, y, z each = 0.001. Find maximum relative error at x = y = z = 1.

  1. Compute partial derivatives at x = y = z = 1:
    ∂u/∂x = 8xy³/z⁴ = 8  |  ∂u/∂y = 12x²y²/z⁴ = 12  |  ∂u/∂z = −16x²y³/z⁵ = −16
  2. Maximum absolute error (take absolute values):
    δu_max ≈ |8|(0.001) + |12|(0.001) + |16|(0.001) = 0.008 + 0.012 + 0.016 = 0.036
  3. Value of u at x = y = z = 1: u = 4(1)(1)/1 = 4
  4. Eᵣ_max = δu / u = 0.036 / 4
  5. Result: Eᵣ_max = 0.009
3 Percentage Error (Eₚ)
Formula
Eₚ = Eᵣ × 100 (%)

Converts the relative error to a percentage. Useful for quick comparison of accuracy across different calculations.

Worked Example 1

Given Eᵣ = 0.000506 (from Example 1 above):

  1. Eₚ = Eᵣ × 100 = 0.000506 × 100
  2. Result: Eₚ = 0.0506%
Worked Example 2

Given Eᵣ = 0.009 (from propagation example above):

  1. Eₚ = 0.009 × 100
  2. Result: Eₚ = 0.9%

🔗 Error Propagation in Functions

For y = f(x₁, x₂, ..., xₙ) with errors δx₁, δx₂, ..., δxₙ in the variables:

General Propagation Formula
δy ≈ |∂y/∂x₁|·δx₁ + |∂y/∂x₂|·δx₂ + ··· + |∂y/∂xₙ|·δxₙ
Special Case: Product y = x₁·x₂·...·xₙ
Eᵣ = δx₁/x₁ + δx₂/x₂ + ··· + δxₙ/xₙ
The relative error of a product equals the sum of the individual relative errors of the factors.
⚠️ Common Errors in Examinations
  • Omitting absolute value signs: Eₐ = |X − X̃|, not X − X̃
  • Dividing by X̃ (approximate) instead of X (true) in the relative error formula
  • Forgetting to multiply by 100 to convert relative error to percentage
  • Confusing rounding error with truncation error
  • In propagation: not taking absolute values of each partial derivative term

🔢 Chapter 2: Solutions of Non-Linear Equations — Foundations

📖 Definitions
Simple Root

α is a simple root if f(α) = 0 and f'(α) ≠ 0.
f(x) = (x − α)g(x), where g(α) ≠ 0.

Multiple Root

α is a root of multiplicity m if f(α) = f'(α) = ... = f^(m-1)(α) = 0 but f^(m)(α) ≠ 0.

This chapter deals with simple roots only, as stated in the lecture notes.
📐 Intermediate Value Theorem — Root Location
Theorem 1 (From Lecture Notes)
If f(x) is continuous on [a, b] and f(a) · f(b) < 0
→ the equation f(x) = 0 has at least one real root in (a, b)

Application: Build a table of f(x) values. Whenever two adjacent values have opposite signs, a root exists between them.

Example from Lecture Notes: f(x) = 8x³ − 12x² − 2x + 3 = 0
x−2−10123
f(x)−105−153−315105

Sign changes occur at intervals: (−1, 0), (0, 1), (1, 2) → 3 real roots

🛑 Stopping Criterion — Used in All Methods
|xₖ₊₁ − xₖ| ≤ ε

Where ε is the tolerance corresponding to the required decimal accuracy. Since infinite iterations cannot be performed, iteration stops when the difference between successive approximations falls below ε.

Alternatively, the criterion |f(xₖ)| ≤ ε may also be used. In practice, the difference criterion is more commonly applied.

⚙️ Four Iterative Methods — Complete Reference

Method 1  Bisection Method (Bolzano Method)

Principle: Based on the repeated application of the Intermediate Value Theorem. The interval containing the root is successively halved, and the sub-interval in which the sign change occurs is retained for the next iteration.

Midpoint Formula
x₁ = (a + b) / 2
Number of Iterations for Accuracy ε
n ≥ ln((b − a) / ε) / ln(2)

Algorithm

  1. Identify interval [a, b] such that f(a) · f(b) < 0
  2. Compute midpoint: x₁ = (a + b) / 2
  3. If f(x₁) = 0: x₁ is the exact root. Stop.
  4. If f(a) · f(x₁) < 0: root lies in [a, x₁]. Set b = x₁.
  5. Else: root lies in [x₁, b]. Set a = x₁.
  6. Repeat from Step 2 until |xₖ₊₁ − xₖ| ≤ ε

Method 2  Method of False Position (Regula Falsi)

Principle: Also called the linear interpolation method. A straight line is drawn between the two bracketing points (xₖ₋₁, fₖ₋₁) and (xₖ, fₖ). The x-intercept of this line is taken as the next approximation.

Iteration Formula
xₖ₊₁ = (xₖ₋₁ · f(xₖ) − xₖ · f(xₖ₋₁)) / (f(xₖ) − f(xₖ₋₁))

For the first step: x₂ = (x₀·f₁ − x₁·f₀) / (f₁ − f₀)

Algorithm

  1. Choose x₀, x₁ such that f(x₀) · f(x₁) < 0
  2. Compute x₂ = (x₀·f₁ − x₁·f₀) / (f₁ − f₀)
  3. If f(x₀) · f(x₂) < 0: root in (x₀, x₂). Keep x₀ fixed, replace x₁ with x₂
  4. Else: root in (x₂, x₁). Keep x₁ fixed, replace x₀ with x₂
  5. Repeat until |xₖ₊₁ − xₖ| ≤ ε
Important: In the method of false position, one endpoint remains fixed throughout all iterations. The fixed endpoint is whichever one maintains the sign change with the new approximation.

Method 3  Newton-Raphson Method (Tangent Method)

Principle: The tangent line to the curve y = f(x) at the current approximation is drawn. The x-intercept of the tangent is taken as the next approximation. Also derived via Taylor series expansion.

⭐ Core Iteration Formula
xₖ₊₁ = xₖ − f(xₖ) / f'(xₖ)   [f'(xₖ) ≠ 0]

Taylor Series Derivation (from Lecture Notes)

  1. Let xₖ + Δx be the exact root, so f(xₖ + Δx) = 0
  2. Expand via Taylor series: f(xₖ) + Δx·f'(xₖ) + (Δx)²/2!·f''(xₖ) + ... = 0
  3. Neglect second and higher powers of Δx: f(xₖ) + Δx·f'(xₖ) ≈ 0
  4. Solve: Δx = −f(xₖ)/f'(xₖ)
  5. Therefore: xₖ₊₁ = xₖ + Δx = xₖ − f(xₖ)/f'(xₖ)

Algorithm

  1. Choose initial approximation x₀ near the root
  2. Compute f(xₖ) and f'(xₖ)
  3. Apply: xₖ₊₁ = xₖ − f(xₖ) / f'(xₖ)
  4. Repeat until |xₖ₊₁ − xₖ| ≤ ε

Method 4  Secant Method

Principle: A secant line is drawn through two most recent approximations. Unlike False Position, no condition f(x₀)·f(x₁) < 0 is required, and both endpoints are updated at each iteration.

General Iteration Formula
xₖ₊₁ = xₖ − f(xₖ) · (xₖ − xₖ₋₁) / (f(xₖ) − f(xₖ₋₁))

Algorithm

  1. Choose two initial approximations x₀ and x₁
  2. Compute f(x₀) and f(x₁)
  3. Apply formula to obtain x₂
  4. Use x₁ and x₂ to compute x₃, and so on (always use the two most recent values)
  5. Stop when |xₖ₊₁ − xₖ| ≤ ε

Key Distinction: Secant Method vs. False Position Method
  • False Position: One endpoint is held fixed throughout; only the other is updated
  • Secant: Both endpoints update at every iteration (always the two most recent points)
  • Both methods use the same secant line formula — the difference lies in which two points are selected

Advantages

  • No derivative required
  • Superlinear convergence
  • No bracket required

Disadvantages

  • Convergence is not guaranteed
  • Fails when f(xₙ) = f(xₙ₋₁)
  • Sensitive to initial guesses

⚡ Problem-Solving Guidelines
Step 1: Root Location

Before applying any method, always construct a table of f(x) values to identify the interval containing the root.

Step 2: Simplify the Formula

For Newton-Raphson, algebraically simplify xₖ₊₁ = xₖ − f/f' into a single fraction before substituting numerical values.

Step 3: Apply Stopping Criterion

Check |xₖ₊₁ − xₖ| ≤ ε at every step. Do not stop prematurely or over-iterate.

Step 4: Identify Fixed Endpoint

In False Position only: determine which endpoint remains fixed. This is a common source of errors in examinations.

Step 5: Deriving Root Formulas

For ⁿ√N: f(x) = xⁿ − N, f'(x) = nxⁿ⁻¹ → xₖ₊₁ = [(n−1)xₖⁿ + N] / [n·xₖⁿ⁻¹]

Step 6: Error in Products

For y = x₁·x₂·...·xₙ: Eᵣ_max = δx₁/x₁ + δx₂/x₂ + ... + δxₙ/xₙ

⚠️ Common Errors — Examination Traps
  • Newton-Raphson: Incorrect differentiation of f(x), especially with composite functions
  • Bisection: Updating the wrong endpoint (update b when f(midpoint) > 0; update a when f(midpoint) < 0)
  • False Position: Treating it like Secant — both endpoints do NOT update; only one does
  • Relative error: Dividing by the approximate value instead of the true value
  • Stopping condition: Using incorrect ε for the required number of decimal places
  • Secant method: The method fails and diverges if f(xₙ) = f(xₙ₋₁)

📋 Quick Revision Sheet

ERROR ESTIMATION — KEY FORMULAS

Eₐ = |X − X̃| Eᵣ = Eₐ / |X| Eₚ = Eᵣ × 100 n decimals → Eₐ ≤ ½×10⁻ⁿ
  • Inherent error: in input data before computation
  • Rounding error: from rounding numbers (round half up)
  • Truncation error: from cutting off an infinite process
  • Relative and percentage errors are independent of units
  • For y = x₁·x₂·...·xₙ: relative error = sum of individual relative errors

ROOT FINDING — PRELIMINARY STEPS

  • Root exists in (a, b) when f(a) · f(b) < 0
  • Construct a table of f(x) at integer or half-integer points
  • Universal stopping condition: |xₖ₊₁ − xₖ| ≤ ε

BISECTION METHOD

Formula: xₙ = (a + b) / 2  ·  Both endpoints update  ·  Always converges  ·  Slow (linear)

f(xₙ) < 0 → new a = xₙ  |  f(xₙ) > 0 → new b = xₙ

FALSE POSITION METHOD (Regula Falsi)

Formula: xₙ₊₁ = (x₀·fₙ − xₙ·f₀) / (fₙ − f₀)  ·  One endpoint is fixed  ·  Always converges  ·  Medium speed

Identify which endpoint is fixed (the one that does not change sign with new approximation)

NEWTON-RAPHSON METHOD ⭐ (Most Important)

Formula: xₖ₊₁ = xₖ − f(xₖ) / f'(xₖ)  ·  Requires derivative  ·  One starting point  ·  Quadratic convergence

Fails when: f'(xₖ) = 0, or starting point is far from root

SECANT METHOD

Formula: xₖ₊₁ = xₖ − f(xₖ)(xₖ − xₖ₋₁) / (f(xₖ) − f(xₖ₋₁))  ·  No derivative  ·  Both points update  ·  Not guaranteed to converge

Fails when f(xₙ) = f(xₙ₋₁). No bracket required.

    f(0.44673) ≈ +0.20354 > 0
  3. Root in (0.44673, 1). x₁ = 1 remains fixed.
    x₄ = (0.44673·(−2.17798) − 1·(0.20354)) / (−2.17798 − 0.20354) ≈ 0.49402

Fixed endpoint: x = 1 (right endpoint)

Converges toward ≈ 0.5178

Question 5 · Hard · Derivation + Application
(a) Derive the Newton-Raphson iteration formula for computing √N.
(b) Apply the formula with x₀ = 2 to compute √5 correct to 4 decimal places.
Part (a) — Derivation:
  1. Let x = √N → x² = N → Define f(x) = x² − N
  2. f'(x) = 2x
  3. Substitute into N-R formula: xₖ₊₁ = xₖ − (xₖ² − N) / (2xₖ)
  4. Simplify: xₖ₊₁ = (2xₖ² − xₖ² + N) / (2xₖ) = (xₖ + N/xₖ) / 2
Part (b) — √5, N = 5, x₀ = 2:
kxₖxₖ₊₁ = (xₖ + 5/xₖ) / 2
02(2 + 2.5)/2 = 2.25
12.25(2.25 + 2.2222)/2 = 2.2361
22.2361(2.2361 + 2.2360)/2 = 2.23607

|x₃ − x₂| < 0.00005 → √5 ≈ 2.2361 ✓
