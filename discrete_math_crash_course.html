LOGIC & MATHEMATICAL PROOF

Propositions · Connectives · Truth Tables · 4 Proof Methods

What is a Proposition?

A declarative sentence that is either TRUE or FALSE — not both, not neither.

✓ Are propositions
✓ "5 + 2 = 8" → FALSE (but it's a proposition) ✓ "17 is prime" → TRUE ✓ "Addis Ababa is in ETH" → TRUE
✗ Are NOT propositions
✗ "What time is it?" → Question ✗ "Go to bed!" → Command ✗ "x + 1 = 4" → Open sentence (depends on x) ✗ "This statement is false" → Paradox

Exam Traps — Propositions

⚠ "x + 1 = 4" is NOT a proposition — truth depends on x
⚠ Questions, commands, exclamations = NOT propositions
⚠ "This statement is false" = paradox = NOT a proposition
✓ If it can be assigned T or F, it's a proposition
✓ Even false statements ARE propositions ("2+2=5" is a proposition)

De Morgan's Laws — Memorize These

¬(P ∨ Q) ≡ ¬P ∧ ¬Q

NOT(P or Q) = (NOT P) and (NOT Q)

¬(P ∧ Q) ≡ ¬P ∨ ¬Q

NOT(P and Q) = (NOT P) or (NOT Q)

Used when writing the contrapositive of statements with "and" or "or"
4 PROOF METHODS  |  Direct · Contrapositive · Contradiction · Induction

Key Definitions — Use These In Every Proof

Even number: n = 2k Odd number: n = 2k + 1 a divides b: b = a·s (s is integer)
Rational number: x = a/b (a,b integers, b≠0) Irrational: NOT expressible as a/b Integer: ..., -2, -1, 0, 1, 2, ...

Method 1: Direct Proof

Use when: Hypothesis P gives concrete algebraic form to work with.

Template
1. Assume P is true
2. Apply definition (even=2k, odd=2k+1, etc.)
3. Calculate / manipulate algebra
4. Show Q follows → conclusion

Direct Proof — Full Example

Prove: if n is odd → n² is odd

Assume: n is odd
Definition: n = 2k + 1 for some integer k
Calculate: n² = (2k+1)² = 4k²+4k+1 = 2(2k²+2k)+1
Observe: 2(2k²+2k)+1 = 2·(integer)+1 = ODD form
∴ Conclude: n² is odd ✓

Method 2: Contrapositive

P→Q is equivalent to ¬Q→¬P. Prove the contrapositive instead.

Use when: Q is easier to negate than P is to work with.

1. Write contrapositive: ¬Q → ¬P
2. Assume ¬Q is true
3. Derive ¬P using algebra/definitions
4. Conclude original P→Q is true

Contrapositive — Full Example

Prove: if n² is odd → n is odd

Original: n² odd → n odd
Contra: n EVEN → n² EVEN
Assume: n is even → n = 2k
Calculate: n² = (2k)² = 4k² = 2(2k²) → EVEN
∴ Conclude: Contrapositive true → original true ✓
⚠ Use De Morgan when negating "A or B" or "A and B"

Method 3: Contradiction

Use when: Statement involves irrationals, primes, or infinities.

1. Assume S is FALSE
2. Work logically from that assumption
3. Arrive at impossible result (contradiction!)
4. Conclude S must be TRUE

Contradiction — √2 is Irrational

Assume: √2 = a/b in lowest terms (b≠0)
Square: 2 = a²/b² → 2b² = a²
Deduction 1: a² even → a even → a = 2c
Substitute: 2b² = 4c² → b² = 2c² → b even
CONTRADICTION: a and b both even → share factor 2 → NOT lowest terms!
∴ Conclude: √2 is irrational ✓

Method 4: Mathematical Induction

Use when: Statement says "For ALL positive integers n, P(n) is true"

2 Required Steps
Step 1 — BASE CASE: Prove P(1) is true (just plug in n=1)
Step 2 — INDUCTIVE STEP:
Assume P(k) is true (Inductive Hypothesis)
Then prove P(k+1) is true using P(k)
Both steps are REQUIRED. Missing either = zero marks.
Example: Prove 1+2+3+...+n = n(n+1)/2
Base (n=1): LHS=1, RHS=1(2)/2=1 ✓
Assume P(k): 1+2+...+k = k(k+1)/2
Prove P(k+1): 1+2+...+k+(k+1)
= P(k)+(k+1): k(k+1)/2 + (k+1)
Factor: (k+1)[k/2 + 1] = (k+1)(k+2)/2
∴ Done: Exactly the formula for n=k+1 ✓
Example 2: Prove n³ − n is divisible by 3
Base (n=1): 1³-1=0 = 3×0 ✓ divisible by 3
Assume P(k): k³-k = 3m for some integer m
P(k+1): (k+1)³-(k+1) = k³+3k²+3k+1-k-1
Simplify: = (k³-k) + 3k²+3k = 3m+3k²+3k
= 3(m+k²+k): divisible by 3 ✓

Which Proof Method? — Quick Guide

Statement typeUse this method
"If P then Q" and P gives algebraic formDirect Proof
"If P then Q" and Q is easier to negateContrapositive
Prove something is irrational, prime, or impossibleContradiction
"For all n ∈ ℕ", sum formulas, divisibility for all nInduction

Divisibility Theorems

If a|b AND a|c → a|(b+c), a|(b-c), a|(mb+nc) If a|b AND b|c → a|c (transitivity) If a|b (b≠0) → |a| ≤ |b|

Division Algorithm

a = bq + r, where 0 ≤ r < b
Example: a=23, b=5 23 = 5×4 + 3 → q=4, r=3 Example: a=-17, b=5 -17 = 5×(-4) + 3 → q=-4, r=3

ELEMENTARY COUNTING PRINCIPLES

Permutations · Combinations · Inclusion-Exclusion · Pigeonhole · Binomial

Rule of Product (Multiplication)

Task 1 done in m ways AND task 2 in n ways → total = m × n

Total = m₁ × m₂ × m₃ × ...
Example: 3 shirt choices × 4 pant choices = 12 outfits Toss a coin 3 times: 2 × 2 × 2 = 8 outcomes
Use when tasks are done TOGETHER (AND)

Rule of Sum (Addition)

Task 1 in m ways OR task 2 in n ways (mutually exclusive) → total = m + n

Total = m + n
Example: Travel by bus (3 routes) OR train (2 routes) = 3 + 2 = 5 ways
Use when choosing ONE of the tasks (OR)

Permutations — Order MATTERS

Arranging r items from n items where order matters. Think: assigning positions, making passwords, ranking.

P(n, r) = n! / (n-r)!
n = total available items r = how many you pick ! = factorial (n! = n×(n-1)×...×2×1) Special: arrange ALL n items P(n,n) = n!
Example
Q: How many 3-letter arrangements from {A,B,C,D,E}?
P(5,3) = 5!/(5-3)! = 5!/2! = 120/2 = 60
Q: Arrange all 4 books on a shelf?
P(4,4) = 4! = 4×3×2×1 = 24

Combinations — Order Does NOT Matter

Selecting r items from n where order doesn't matter. Think: teams, committees, groups.

C(n, r) = n! / [r! × (n-r)!]
Also written as C(n,r) or "n choose r" Relationship to Permutation: P(n,r) = C(n,r) × r!
Example
Q: Choose a committee of 3 from 7 people?
C(7,3) = 7!/(3!×4!) = 5040/(6×24) = 35
Q: Choose 2 players from 10 for a doubles match?
C(10,2) = 10!/(2!×8!) = 90/2 = 45

Permutation vs Combination — THE KEY DECISION

Question says...Keyword clueUse
"How many arrangements/sequences/orders"arrange, rank, assign position, passwordPermutation P(n,r)
"How many selections/groups/committees"choose, select, group, team, committeeCombination C(n,r)
"How many ways to assign specific roles"president/VP/treasurer (specific roles)Permutation P(n,r)
"How many ways to pick a team"team of 5 (no specific roles)Combination C(n,r)
Golden Rule: Does ORDER matter? YES → Permutation Does ORDER matter? NO → Combination

Inclusion-Exclusion Principle

When sets overlap, simple addition double-counts. Subtract the intersection.

For 2 sets
|A ∪ B| = |A| + |B| - |A ∩ B|
For 3 sets
|A∪B∪C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|
Worked Example
Given: 30 students. 18 play football, 15 basketball, 8 play both.
Find: How many play at least one sport?
|F∪B| = 18 + 15 - 8 = 25
⚠ ALWAYS subtract the intersection. Forgetting -|A∩B| is the #1 exam mistake.

Pigeonhole Principle

If more items than containers → at least one container has 2+ items.

Simple: n+1 items in n containers → at least 1 container has ≥ 2 items Generalized: N items in k containers → at least 1 container has ≥ ⌈N/k⌉ items (⌈ ⌉ = ceiling = round UP)

Pigeonhole Examples

Q: Among 13 people, must 2 share a birth month?
Pigeons=13 people, Holes=12 months. 13 > 12 → YES ✓
Q: Among 5 integers, must 2 have same remainder mod 4?
Remainders mod 4: {0,1,2,3} = 4 options. 5 > 4 → YES ✓
Identify: WHAT are the pigeons? WHAT are the holes?

Binomial Theorem

(x+y)ⁿ = Σ C(n,k) · x^(n-k) · y^k (k from 0 to n)
Specific term formula
T(r+1) = C(n,r) · x^(n-r) · y^r
Example: Expand (x+y)³
= C(3,0)x³ + C(3,1)x²y + C(3,2)xy² + C(3,3)y³ = 1·x³ + 3·x²y + 3·xy² + 1·y³ = x³ + 3x²y + 3xy² + y³
Coefficients from Pascal's triangle: 1, 3, 3, 1

PROBABILITY

Sample Space · Axioms · Conditional · Bayes' · Random Variables · Expected Value

What is Probability?

Probability measures how likely an event is to occur. It is always a number between 0 and 1.

Sample Space (S) = set of ALL possible outcomes Event (E) = a subset of S P(E) = probability of event E 0 ≤ P(E) ≤ 1 P(S) = 1 (something always happens) P(∅) = 0 (impossible event)
Basketball analogy: S = all possible shots. E = shots that go in. P(E) = fraction that score.

Key Definitions

TermMeaning
ExperimentAny process with uncertain outcome
Sample Space SAll possible outcomes
Event EA subset of S
OutcomeA single result of an experiment
Complement EᶜAll outcomes NOT in E
Example: Roll a die S = {1, 2, 3, 4, 5, 6} E = "even number" = {2, 4, 6} P(E) = 3/6 = 1/2

Classical (Equally Likely) Probability

P(E) = |E| / |S| = (favorable outcomes) / (total outcomes)
⚠ Only valid when all outcomes are EQUALLY likely
Examples
Toss a coin: P(Heads) = 1/2
Roll a die, get 4: P(4) = 1/6
Pick a card, get Ace: P(Ace) = 4/52 = 1/13
Pick 1 from {1..10}, get even: P = 5/10 = 1/2

Probability Axioms & Rules

3 Kolmogorov Axioms
Axiom 1: P(E) ≥ 0 (non-negative) Axiom 2: P(S) = 1 (certain event) Axiom 3: If E∩F = ∅ then P(E∪F) = P(E) + P(F) (mutually exclusive)
Complement Rule
P(Eᶜ) = 1 − P(E)
If P(rain) = 0.3, then P(no rain) = 0.7
Addition Rule (General)
P(E∪F) = P(E) + P(F) − P(E∩F)
Mutually Exclusive Events
If E∩F = ∅: P(E∪F) = P(E) + P(F) (no overlap)
Worked Example
P(A) = 0.4, P(B) = 0.3, P(A∩B) = 0.1
P(A∪B) = 0.4 + 0.3 − 0.1 = 0.6

Conditional Probability

Probability of E given that F already happened. F shrinks your sample space.

P(E | F) = P(E ∩ F) / P(F), P(F) ≠ 0
Multiplication Rule (rearranged)
P(E ∩ F) = P(E | F) · P(F) = P(F | E) · P(E)

Conditional — Worked Example

Experiment: Roll a die. E = "get 4", F = "get even number"
P(F) = 3/6 = 1/2
P(E ∩ F) = P({4}) = 1/6
P(E | F) = (1/6) / (1/2) = 2/6 = 1/3
Meaning: Among even outcomes {2,4,6}, only 1 is "4" → 1/3 ✓
⚠ P(E|F) ≠ P(F|E) — order matters!

Bayes' Theorem

Reverses conditional probability. Given P(E|F), find P(F|E).

Simple Form
P(F | E) = P(E | F) · P(F) / P(E)
Full Form (with Total Probability)
P(F | E) = P(E|F)·P(F) / [P(E|F)·P(F) + P(E|Fᶜ)·P(Fᶜ)]
Total Probability Law
P(E) = P(E|F)·P(F) + P(E|Fᶜ)·P(Fᶜ)
Worked Example
Given: Disease test. P(Disease) = 0.01
P(Positive | Disease) = 0.99
P(Positive | No Disease) = 0.05
Find: P(Disease | Positive)
P(Pos) = 0.99×0.01 + 0.05×0.99 = 0.0099 + 0.0495 = 0.0594
P(Disease|Pos) = (0.99 × 0.01) / 0.0594
= 0.0099 / 0.0594 ≈ 0.1667 = 16.67%

Independent Events

Two events are independent if knowing one happened gives NO information about the other.

E and F are independent if: P(E ∩ F) = P(E) · P(F) Equivalently: P(E | F) = P(E) P(F | E) = P(F)
⚠ Independent ≠ Mutually Exclusive!
Mutually exclusive: P(E∩F)=0
Independent: P(E∩F)=P(E)·P(F)

Independence — Examples

Toss coin twice: H on 1st and H on 2nd are independent
P(H₁ ∩ H₂) = 1/2 × 1/2 = 1/4 = P(H₁)·P(H₂) ✓
Check independence: P(A)=0.4, P(B)=0.3, P(A∩B)=0.12
P(A)·P(B) = 0.4×0.3 = 0.12 = P(A∩B) → INDEPENDENT ✓
n Independent events:
P(E₁ ∩ E₂ ∩...∩ Eₙ) = P(E₁)·P(E₂)···P(Eₙ)

Random Variables

A random variable X assigns a number to each outcome in the sample space.

Discrete Random Variable

Takes countable values (like 0, 1, 2, 3...)

Probability Distribution: Lists all values x and their probabilities P(X=x) Requirements: P(X=x) ≥ 0 for all x Σ P(X=x) = 1 (must sum to 1)
Example: Toss 2 coins, X = number of Heads
x012
P(X=x)1/42/41/4
Cumulative Distribution F(x)
F(x) = P(X ≤ x) = Σ P(X=k) for k ≤ x
Example (continued)
xF(x) = P(X≤x)
01/4 = 0.25
11/4 + 2/4 = 3/4 = 0.75
23/4 + 1/4 = 4/4 = 1
F(x) always ends at 1 (by the last value)

Expected Value & Variance

Expected Value (Mean) E(X)
E(X) = Σ x · P(X = x)

The long-run average value of X.

Variance Var(X)
Var(X) = E(X²) − [E(X)]²
Where: E(X²) = Σ x² · P(X = x)
Standard Deviation
σ = √Var(X)
Full Worked Example
X = number of Heads in 2 coin tosses
xP(X=x)x·Px²·P
01/400
12/42/42/4
21/42/44/4
Σ14/4 = 16/4 = 1.5
E(X) = 1
Var(X) = 1.5 − 1² = 0.5
σ = √0.5 ≈ 0.707
Properties of E(X)
E(aX + b) = a·E(X) + b E(X + Y) = E(X) + E(Y) (always) E(X · Y) = E(X)·E(Y) (only if independent)
Properties of Var(X)
Var(aX + b) = a²·Var(X) Var(X + Y) = Var(X) + Var(Y) (if independent)
